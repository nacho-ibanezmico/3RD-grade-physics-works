# Undergraduate Computational Physics & AI Portfolio

Welcome to my repository containing computational projects, simulations, and data analyses developed during my third year of Physics. This portfolio showcases the intersection of advanced physics, applied mathematics, and modern machine learning techniques.

## 📂 Repository Structure

The code is organized into distinct directories based on core physics disciplines and data science methodologies:

###  Data Science & Artificial Intelligence
*   **`Introduction to AI/`** – Implementations of machine learning algorithms, optimization techniques, and foundational AI models applied to scientific problems.
*   **`Exploratory Data Analysis/`** – Statistical analysis, data cleaning, and advanced visualizations used to extract insights from complex datasets.

### Theoretical & Statistical Physics
*   **`Quantum Mechanics/`** – Numerical solutions to the Schrödinger equation, wave function simulations, and computational matrix mechanics.
*   **`Statistical Physics/`** – Simulations of thermodynamic systems, particle distributions, and stochastic processes (e.g., Monte Carlo methods).

### Applied Physics & Mathematics
*   **`Electronic Physics/`** – Computational modeling of circuits, semiconductor behavior, or electronic signal processing.
*   **`Differential Geometry/`** – Visualization and numerical analysis of manifolds, curvature, and geometric tensors.

## Tech Stack & Libraries
*   **Language:** Python 3.x
*   **Environment:** Visual Studio Code (VS Code) & Jupyter Notebooks (`.ipynb`)
*   **Core Libraries:** NumPy, SciPy, Matplotlib, Pandas, Scikit-Learn *(Note: Feel free to adjust these based on what you used!)*

## How to View the Notebooks
GitHub natively renders Jupyter Notebooks, but large files can sometimes fail to load. If a notebook doesn't display correctly, you can copy its URL and paste it into [Jupyter nbviewer](https://nbviewer.org/) for a guaranteed, clean render.
